# PHP Bug Hunt Challenge

Welcome! This repository contains a simple PHP script used for calculating shopping cart totals. However, the calculation is currently broken and it's throwing a warning.

## Your Tasks:
1. Clone this repository and run `calculate.php` locally. 
2. Create a new branch named `fix/discount-logic`.
3. **Task 1:** Find and fix the undefined variable error. Commit this change on its own (e.g., `git commit -m "fix: remove undefined tax variable"`).
4. **Task 2:** Find and fix the mathematical logic error so the discount applies correctly. Commit this change on its own.
5. Push your branch to GitHub and open a **Pull Request** against the `main` branch.
6. In your PR description, briefly explain what the two bugs were and how you fixed them.

Good luck!
